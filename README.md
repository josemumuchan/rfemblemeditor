# RF Emblem Editor
RF Emblem Editor is a tool to edit guild emblems in the game RF Online, developed with Claude AI.

Use the tool here: https://josemumuchan.github.io/rfemblemeditor/

**How to:**

👉 [Clear all except your guild](#clear-all-except-your-guild)

👉 [Replace emblem with custom](#replace-emblem-with-custom)

# Usage:
1. Select emblem.spr to load
2. Choose which icon to "CLEAR" or "REPLACE", Atlas 1 = guild emblem background, Atlas 2 = guild emblem icon
3. Select all to select all icons, or select your guild emblem and click "Invert" to choose all other emblems
4. Save SPR and move it to your RF Folder\SpriteImage\common

# Clear all except your guild

https://github.com/user-attachments/assets/57ae0fd1-3301-4e3b-9b54-57948d93adc9

1. Select emblem.spr to load
2. Select your guild emblem in Atlas 1
3. Click "Invert"
4. Click "CLEAR TILES"
5. Repeat steps 2-5 in Atlas 2
6. Save SPR and move it to your RF Folder\SpriteImage\common

# Replace emblem with custom

https://github.com/user-attachments/assets/8fcfb568-6931-4a6c-8e3d-ee5ce7acb71e

1. Select emblem.spr to load
2. Select your guild emblem in Atlas 2
3. Select your guild emblem
4. Select your custom emblem or drag and drop
5. Click "APPLY"
6. Save SPR and move it to your RF Folder\SpriteImage\common
